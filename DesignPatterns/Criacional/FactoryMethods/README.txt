O FactoryMethod substitui as chamadas diretas de construção de objetos da classe(new), 
para um método intermediarioque fara o papel de fabrica, este factory method(método fabrica), irá controlar como criar os objetos que estendem 
a sua classe serão instanciados.
Para analisar esse exemplo em execução utilize umas dessas opções(uber, log, eats), exemplo: npm run dev -- --uber
